# Marvel Cinematic Universe Landing Page

A modern, responsive landing page inspired by the Marvel Cinematic Universe (MCU). The website showcases the journey of the MCU through its major phases, iconic heroes, and the Multiverse Saga using a cinematic design with smooth scrolling and animated sections.

---

## Features

- Responsive landing page design
- Modern navigation bar
- Full-screen hero section
- Timeline showcasing MCU phases
- Character showcase with image cards
- Multiverse promotional section
- Animated scroll effects
- Smooth user experience
- Clean and organized HTML structure

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Fonts
  - Oswald
  - Inter

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/marvel-landing-page.git
```

2. Navigate to the project folder

```bash
cd marvel-landing-page
```

3. Open `index.html` in your browser.

No additional dependencies or build tools are required.
